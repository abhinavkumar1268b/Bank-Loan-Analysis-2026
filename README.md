**📊 Financial Loan Analysis Dashboard**

_**📌 Project Overview**_

This project focuses on analyzing financial loan data to uncover key business insights and monitor loan performance. The analysis pipeline involves data cleaning, SQL-based querying, Python-driven analysis, and the creation of an interactive Power BI dashboard for real-time visualization and decision-making.

_**🚀 Features**_
Data cleaning and preprocessing using Microsoft Excel.
Data extraction and analysis with Microsoft SQL Server.
Exploratory Data Analysis (EDA) using Python.
Real-time interactive dashboard built with Power BI.
Analysis of:
Total Loan Applications
Funded Amount
Amount Received
Good Loan vs Bad Loan Performance
Loan Status Distribution
Monthly Trends
Regional Analysis
Purpose-wise Loan Analysis
Interest Rate and DTI Metrics

_**🛠️ Technologies Used**_

Microsoft Excel	Data Cleaning and Preparation
Microsoft SQL Server	Querying and Data Analysis
Python	Exploratory Data Analysis and Insights
Power BI	Interactive Dashboard and Visualization
